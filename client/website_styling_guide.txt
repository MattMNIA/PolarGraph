# Website Styling Guide: Matthew Morgan's Portfolio

## Overview
This is a modern, minimalist portfolio website built with React, TailwindCSS, and Framer Motion. The design features a clean, professional aesthetic with a focus on readability, smooth animations, and a sophisticated dark/light mode toggle. The site uses a blue accent color scheme and maintains consistent spacing and typography throughout.

## Theme System

### ThemeProvider Setup
```javascript
// ThemeProvider.js
import React, { createContext, useContext, useState, useEffect } from 'react';

const ThemeContext = createContext();

export const ThemeProvider = ({ children }) => {
  const [darkMode, setDarkMode] = useState(() => {
    const saved = localStorage.getItem('theme');
    return saved ? JSON.parse(saved) : true; // Default to dark
  });

  useEffect(() => {
    const root = document.documentElement;
    if (darkMode) {
      root.classList.add('dark');
    } else {
      root.classList.remove('dark');
    }
    localStorage.setItem('theme', JSON.stringify(darkMode));
  }, [darkMode]);

  const toggleTheme = () => setDarkMode(!darkMode);

  return (
    <ThemeContext.Provider value={{ darkMode, toggleTheme }}>
      {children}
    </ThemeContext.Provider>
  );
};

export const useTheme = () => useContext(ThemeContext);
```

  return `${baseClasses} ${darkMode ? themeClasses.dark : themeClasses.light}`;

## Key Design Principles

1. **Color Scheme**: Blue primary (#3b82f6), gray backgrounds, white/dark text
2. **Typography**: Clean sans-serif, large headings, good line spacing
3. **Spacing**: Consistent 24px section padding, 6xl max-width containers
4. **Shadows**: Subtle shadows on cards and buttons
5. **Animations**: Smooth fade-in-up animations, hover effects
6. **Responsive**: Mobile-first design with responsive grids
7. **Dark Mode**: Comprehensive dark mode support with theme persistence
8. **Accessibility**: Semantic HTML, proper contrast ratios, keyboard navigation

## Implementation Notes

- Use the theme system consistently across all components
- Apply `getThemeClasses` for all theme-aware styling
- Include Framer Motion animations on scroll-triggered elements
- Maintain the blue accent color for interactive elements
- Use the container class for consistent max-widths
- Implement smooth transitions for all interactive elements
- Ensure all components work in both light and dark modes

This styling system creates a cohesive, professional portfolio that can be easily extended with new sections or pages while maintaining visual consistency.
